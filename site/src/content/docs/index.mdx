**LibGen MCP** is an [MCP](https://modelcontextprotocol.io) server, written in Go, for searching and downloading from **Library Genesis** (the `libgen.li` mirror family). It exposes three tools — `search`, `get_details`, and `download` — to any MCP-compatible client such as Claude Code, Claude Desktop, or your own agent.

Mirrors are discovered automatically and cached, with transparent failover, so the server keeps working as individual mirrors go up and down. Articles can also be fetched from open-access and Sci-Hub sources by DOI.

## Frequently asked questions

### What is libgen-mcp?

libgen-mcp is an open source Model Context Protocol (MCP) server, written in Go, for searching and downloading from Library Genesis (the `libgen.li` mirror family). It exposes three tools — `search`, `get_details`, and `download` — to any MCP-compatible client such as Claude Code, Claude Desktop, or your own agent, so you can search books and resolve download links in plain language.

### What can libgen-mcp do?

It searches the Library Genesis catalog, returns per-result metadata and MD5 hashes, fetches full details for a record, and downloads files to a local directory. Books are downloaded by MD5 from LibGen, and articles are fetched by DOI from open-access (Unpaywall) and Sci-Hub sources. Mirrors are discovered automatically and cached, with transparent failover, retry with cooldown, and a shared rate limiter, so the server keeps working as individual mirrors go up and down.

### Do I need an account, API key, or token to use libgen-mcp?

No. libgen-mcp requires no Library Genesis account, no API key, and no token. Every option is an environment variable with a documented default, so the server runs with zero configuration. The only email-shaped setting is the optional `LIBGEN_MCP_UNPAYWALL_EMAIL` contact address sent to the Unpaywall API for DOI lookups. It is unset by default, which disables the Unpaywall source (Sci-Hub still serves DOIs without it); set your own address to enable Unpaywall.

### Is libgen-mcp free and open source?

Yes. libgen-mcp is free and open source under the MIT license. The full source code is on GitHub at [github.com/jmrplens/libgen-mcp](https://github.com/jmrplens/libgen-mcp), and prebuilt binaries, a Docker image, and a Claude Desktop extension are published with each release.
