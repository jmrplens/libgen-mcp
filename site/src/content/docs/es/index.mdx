**LibGen MCP** es un servidor [MCP](https://modelcontextprotocol.io), escrito en Go, para buscar y descargar desde **Library Genesis** (la familia de mirrors `libgen.li`). Expone tres herramientas — `search`, `get_details` y `download` — a cualquier cliente compatible con MCP como Claude Code, Claude Desktop o tu propio agente.

Los mirrors se descubren automáticamente y se cachean, con failover transparente, de modo que el servidor sigue funcionando aunque algunos mirrors caigan o vuelvan. Los artículos también pueden obtenerse por DOI desde fuentes de acceso abierto y Sci-Hub.

## Preguntas frecuentes

### ¿Qué es libgen-mcp?

libgen-mcp es un servidor Model Context Protocol (MCP) de código abierto, escrito en Go, para buscar y descargar desde Library Genesis (la familia de mirrors `libgen.li`). Expone tres herramientas — `search`, `get_details` y `download` — a cualquier cliente compatible con MCP como Claude Code, Claude Desktop o tu propio agente, para que puedas buscar libros y resolver enlaces de descarga en lenguaje natural.

### ¿Qué puede hacer libgen-mcp?

Busca en el catálogo de Library Genesis, devuelve metadatos y hashes MD5 por resultado, obtiene los detalles completos de un registro y descarga archivos a un directorio local. Los libros se descargan por MD5 desde LibGen, y los artículos se obtienen por DOI desde fuentes de acceso abierto (Unpaywall) y Sci-Hub. Los mirrors se descubren automáticamente y se cachean, con failover transparente, reintentos con enfriamiento y un limitador de tasa compartido, de modo que el servidor sigue funcionando aunque algunos mirrors caigan o vuelvan.

### ¿Necesito una cuenta, clave de API o token para usar libgen-mcp?

No. libgen-mcp no requiere ninguna cuenta de Library Genesis, ninguna clave de API ni ningún token. Cada opción es una variable de entorno con un valor por defecto documentado, así que el servidor funciona sin configuración alguna. El único ajuste con forma de email es la dirección de contacto opcional `LIBGEN_MCP_UNPAYWALL_EMAIL` que se envía a la API de Unpaywall para las consultas por DOI, y viene con un valor por defecto que funciona.

### ¿libgen-mcp es gratis y de código abierto?

Sí. libgen-mcp es software libre y de código abierto bajo la licencia MIT. El código fuente completo está en GitHub en [github.com/jmrplens/libgen-mcp](https://github.com/jmrplens/libgen-mcp), y con cada release se publican binarios precompilados, una imagen de Docker y una extensión para Claude Desktop.
